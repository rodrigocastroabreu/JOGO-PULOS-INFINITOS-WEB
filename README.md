# JOGO PULOS INFINITOS WEB

Este projeto é a versão HTML5 exportada do jogo "Pulos Infinitos".
Ele inclui todos os recursos necessários para rodar o jogo em navegadores compatíveis com HTML5.

## Sobre este jogo

O jogo Pulos Infinitos é um jogo de aventura no qual o personagem Super Bola Ultimate usa seus poderes para completar desafios e pegar medalhas de bronze, prata, ouro e a medalha 3RN Games Entretenimento.

O jogo possui vários obstáculos, espinhos, estrelas e outros itens. O objetivo é chegar o mais longe possível e ver quantos metros você já percorreu. Existem pontos de salvamento no jogo para continuar a partida.

### Os poderes da Super Bola Ultimate

- Prêmio A: Força do pulo de 1520 pixels por segundo e escala 0.5
- Prêmio D: Força do pulo de 1100 pixels por segundo e escala 0.1
- Prêmio N: Força do pulo de 1725 pixels por segundo e escala 0.4
- Prêmio S: Força do pulo de 1850 pixels por segundo e escala 1.2
- Prêmio R: Força do pulo de 1400 pixels por segundo e escala 0.2
- Prêmio 3RN GAMES: Força do pulo de 1850 pixels por segundo e escala 2.2

### Detalhes do aplicativo

- Resolução de tela FULL HD 1920x1080
- Escolha cor do personagem

- Teste suas habilidades

## Estrutura do projeto

- `index.html` - página inicial do jogo
- `style.css` - estilos de layout e interface
- `sw.js` - service worker para suporte offline
- `offline.json` - configurações de cache e recursos offline
- `data.json` - dados de configuração do jogo
- `workermain.js` - código principal para worker de jogos
- `scripts/` - scripts auxiliares e runtime do Construct 3
- `icons/`, `images/`, `media/` - recursos gráficos e de áudio usados no jogo
